# deploy

   

 1. Install [Heroku Toolbelt](https://toolbelt.heroku.com/)
 2. `heroku login`
 3. `heroku git:remote -a damp-oasis-51013`
 4. `git push heroku master`
 5. Go to [https://damp-oasis-51013.herokuapp.com](https://damp-oasis-51013.herokuapp.com)
